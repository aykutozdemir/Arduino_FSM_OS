Detailed Description

Mutex for cooperative task synchronization.

Mutex provides mutual exclusion for shared resources in a cooperative multitasking environment. Only one task can hold the mutex at a time.

Note
This mutex is designed for cooperative multitasking only. It does not provide blocking behavior - tasks must check try_lock() and yield if the mutex is not available.
