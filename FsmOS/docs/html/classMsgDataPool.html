Detailed Description

Memory pool for efficient MsgData allocation.

MsgDataPool manages a pool of MsgData objects to avoid frequent memory allocation/deallocation. It uses an adaptive limit system to balance memory usage and performance.

Note
The pool automatically adjusts its size based on usage patterns.
