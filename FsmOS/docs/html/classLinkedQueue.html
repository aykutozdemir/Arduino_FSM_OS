Detailed Description

template<typename T>
class LinkedQueue< T >

Lightweight, interrupt-safe linked queue.

LinkedQueue provides a simple queue implementation that is safe to use from interrupt context. It uses a linked list structure for efficient insertion and removal.

Template Parameters
TType of elements to store in the queue
Note
This queue is not thread-safe for concurrent access
