Detailed Description

Semaphore for resource counting and synchronization.

Semaphore allows a limited number of tasks to access a resource simultaneously. It maintains a count of available resources.

Note
This semaphore is designed for cooperative multitasking only. Tasks must check wait() and yield if no resources are available.
