Detailed Description

Smart pointer-like wrapper for MsgData with reference counting.

SharedMsg provides automatic memory management for messages through reference counting. When the last reference is destroyed, the message is automatically returned to the pool.

Note
This class is thread-safe for single-threaded cooperative multitasking.
Warning
Do not use across different scheduler instances.
