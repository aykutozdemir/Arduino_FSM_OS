\doxysubsection{Detailed Description}
\doxylink{classSemaphore}{Semaphore} for resource counting and synchronization. 

\doxylink{classSemaphore}{Semaphore} allows a limited number of tasks to access a resource simultaneously. It maintains a count of available resources.

\begin{DoxyNote}{Note}
This semaphore is designed for cooperative multitasking only. Tasks must check \doxylink{classSemaphore_ab733c9bf26e7dd7e007c68ca27b8c06e}{wait()} and yield if no resources are available. 
\end{DoxyNote}
