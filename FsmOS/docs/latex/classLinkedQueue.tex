class Linked\+Queue$<$ T $>$}
Lightweight, interrupt-\/safe linked queue. 

\doxylink{classLinkedQueue}{Linked\+Queue} provides a simple queue implementation that is safe to use from interrupt context. It uses a linked list structure for efficient insertion and removal.
