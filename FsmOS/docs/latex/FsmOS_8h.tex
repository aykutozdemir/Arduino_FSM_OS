\doxysubsection{Detailed Description}
A lightweight cooperative task scheduler for Arduino. 

\begin{DoxyAuthor}{Author}
Aykut Özdemir \href{mailto:aykutozdemirgyte@gmail.com}{\texttt{ aykutozdemirgyte@gmail.\+com}} 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025-\/10-\/02
\end{DoxyDate}
Fsm\+OS provides a simple, memory-\/efficient task scheduler for Arduino, supporting cooperative multitasking, message passing, and system monitoring.

Key features\+:
\begin{DoxyItemize}
\item Cooperative task scheduling with configurable periods
\item Inter-\/task communication through messages and events
\item Dynamic task creation and deletion
\item Memory management and monitoring
\item System diagnostics and profiling
\item Logging system with multiple levels
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This library is designed for AVR-\/based Arduino boards but includes partial support for other architectures.
\end{DoxyNote}
