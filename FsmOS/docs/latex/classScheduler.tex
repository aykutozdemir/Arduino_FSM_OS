\doxysubsection{Detailed Description}
Core scheduler and task manager for Fsm\+OS. 

\doxylink{classScheduler}{Scheduler} manages the execution of tasks, message routing, and system resources. It provides the main interface for task management and system control.

\begin{DoxyNote}{Note}
Only one scheduler instance should exist per application. The global OS instance is provided for convenience.
\end{DoxyNote}
