\doxysubsection{Detailed Description}
\doxylink{classMutex}{Mutex} for cooperative task synchronization. 

\doxylink{classMutex}{Mutex} provides mutual exclusion for shared resources in a cooperative multitasking environment. Only one task can hold the mutex at a time.

\begin{DoxyNote}{Note}
This mutex is designed for cooperative multitasking only. It does not provide blocking behavior -\/ tasks must check try\+\_\+lock() and yield if the mutex is not available. 
\end{DoxyNote}
