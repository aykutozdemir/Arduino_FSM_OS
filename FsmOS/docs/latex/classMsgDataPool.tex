\doxysubsection{Detailed Description}
Memory pool for efficient Msg\+Data allocation. 

\doxylink{classMsgDataPool}{Msg\+Data\+Pool} manages a pool of Msg\+Data objects to avoid frequent memory allocation/deallocation. It uses an adaptive limit system to balance memory usage and performance.

\begin{DoxyNote}{Note}
The pool automatically adjusts its size based on usage patterns. 
\end{DoxyNote}
